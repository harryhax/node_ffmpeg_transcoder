# Video Encoding Auditor Release

This archive contains the distributable version of the Video Encoding Auditor & Plex Transcoder tool.

## Included Files
- All source code (Node.js, Express, frontend, CLI tools)
- README.md with user-friendly instructions
- package.json and dependencies
- Public assets (HTML, JS, CSS)

## How to Use
1. Extract the archive to your server or local machine.
2. Run `npm install` to install dependencies.
3. Start the web UI with `npm run server`.
4. Open http://localhost:3000 in your browser.

## Requirements
- Node.js 18+
- ffmpeg and ffprobe in your PATH
